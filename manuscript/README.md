# manuscript/

A short descriptive paper **about** the newspaper layout engine. It exists so
the repository pipeline's Stage 03 (infrastructure manuscript renderer) produces
a standard PDF for this project, exactly like the sibling templates.

> Do not confuse this with the newspaper. The 12-page *Triplicate* edition is
> defined in `../content/` and rendered by `../scripts/20_render_newspaper.py`
> into `../output/pdf/the-triplicate.pdf`.

Sections: `00_abstract` · `01_introduction` · `02_engine_architecture` ·
`03_typography_and_figures` · `04_reproducibility` · `99_references`.
